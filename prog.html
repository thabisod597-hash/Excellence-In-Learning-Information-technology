1. What is Structured Programming?

Structured programming is a way of writing programs that are:
a, Easy to read.
b, Easy to understand.
c, Easy to test and fix.
d, Organized using functions, decisions, and loops.

2. Variables

A variable is a named memory location used to store data.

Syntax of declaring a variable

a: data_type variable_name;

Example

int age;
float temperature;
char grade;

Common Data Types

Data Type Meaning Example
int Whole numbers int age = 20;
float Decimal numbers float temp = 36.5;
double More accurate decimal numbers double price = 99.99;
char One character char grade = 'A';
bool True or False bool pass = true;

Initialising variables

Giving a variable a data type

Example

a) int age;
age = 17;

b) float temperature;
temperature=34.12;

c) char grade;
grade = 'T';

d) int num;
num = 1;

Modifiers

a;unsigned → stores only positive integers.
b;long → stores very large integers. 3.

Errors in Programming

Programming errors are mistakes that stop a program from working correctly.

(a) Syntax Error

Mistakes in writing code.
Example:
int x = 10
❌ Missing semicolon.

(b) Runtime Error

Occurs while the program is running.
Example:
10 / 0

(c) Logic Error

The program runs but gives the wrong answer.
Example:
2 + 2 = 5

a;Single-line comment:

// This is a comment

b;Multi-line comment:

/* This is

5. Control Flow

Control flow is the order in which program instructions are executed.

It helps a program:

a;Make decisions.
b;Repeat tasks.
c;Follow a logical sequence.

6. Pseudocode

Pseudocode is a simple way of writing program steps using everyday English.
Example:
:Start :Input age If age >= 18
Display "Adult"
Else
Display "Child"
End

Advantages:

a;Easy to understand.
b;Does not use programming language syntax.
c;Helps plan a program before coding.

7.Flowchart

A flowchart is a picture that shows how a program works.

Symbols:

⭕ Oval → Start/End
▭ Rectangle → Process
◇ Diamond → Decision
→ Arrow → Direction of flow

8.Control Structures

Control structures control how a program executes.

They include:

Decision statements
Loops

9.Decision Statements

Decision statements choose between different actions.

a.if Statement

Runs code only if a condition is true.

Example:

if (marks >= 50)
{
cout << "Pass";
}

b.if-else Statement

Chooses between two actions.

Example:

if (marks >= 50)
{
cout << "Pass";
}
else
{
cout << "Fail";
}

c.switch Statement

Used when there are many choices based on one variable.

Example:

switch(choice)
{
case 1:
cout<<"Home";
break;
case 2:
cout<<"Help";
break;
default:
cout<<"Invalid";
}

10.Loops

Loops repeat code.
a.For Loop
Used when you know how many times to repeat.

Example:

for(int i=1; i<=5; i++)
{
cout< }
b.While Loop
Repeats while a condition is true.

Example:

while(x<10)
{
x++;
}

11. Functions

A function is a reusable block of code that performs one task.

Benefits:

a;Reduces repetition.
b;Makes programs easier to read. c;Makes debugging easier.

Types of Functions

a.Pre-defined Functions
Already provided by the programming language.

Examples:

sqrt()
pow()
strlen()
b.User-defined Functions
Created by the programmer.

Example:

int add(int a, int b)
{
return a+b;
}

12.Referencing Functions

Functions can:

a;Be called from other functions. b;Receive information (parameters). c;Return results. This helps make programs more flexible and reusable.
