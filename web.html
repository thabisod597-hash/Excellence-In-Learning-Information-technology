Web Design Notes:

Creating a Login and Registration Page (HTML & CSS)

Introduction

A login page allows registered users to access a website by entering their username/email and password.

A registration page allows new users to create an account by entering their personal information.
CSS (Cascading Style Sheets) is used to make these pages attractive, organized, and easy to use. HTML Structure of a Login Page
A basic login page contains:
1.form> – Holds all input fields.
2.label> – Describes each input.
3.input> – Allows users to enter information.
4.button> – Submits the form.
5.a> – Creates links such as "Forgot Password?" or "Register".

Example:

HTML
a)form>
b)label>Username label>
c)input type="text">
d)label>Password label>
e)input type="password">
f)button>Login button>
g)form>
#)HTML Structure of a Registration Page

A registration page usually contains:

1)Full Name
2)Username
3)Email
4)Password
5)Confirm Password
6)Gender
7)Date of Birth
8)Phone Number
9)Register Button

Example:

HTML
a)form>
b)input type="text" placeholder="Full Name">
c)input type="email" placeholder="Email">
d)input type="password" placeholder="Password">
e)input type="password" placeholder="Confirm Password">
f)button>Register g)/form>

CSS(Cascading Style Sheets)

CSS controls the appearance of a webpage.

CSS syntax:

CSS
selector{
property:value;
}

Example:

CSS
h1{
color:blue;
}
Important CSS Properties

1. color

Changes text colour.
CSS
color:white;

2. background-color

Changes the background colour.
CSS
background-color:blue;

3. width

Sets the width.
CSS
width:300px;

4. height

Sets the height.
CSS
height:40px;

5. border

Creates a border.
CSS
border:1px solid black;

6. border-radius

Rounds corners.
CSS
border-radius:10px;
For a circle:
CSS
border-radius:50%;

7. padding

Creates space inside an element.
Example:
CSS
padding:10px;
Different sides:
CSS
padding-top:10px;
padding-bottom:10px;
padding-left:20px;
padding-right:20px;

8. margin

Creates space outside an element.
CSS
margin:20px;
Center a form:
CSS
margin:auto;

9. font-size

Changes text size.
CSS
font-size:20px;

10. font-family

Changes the font style.
CSS
font-family:Arial;

11. text-align

Aligns text.
CSS
text-align:center;

12. box-shadow

Adds a shadow.
CSS
box-shadow:0 0 10px gray;

13. cursor

Changes the mouse pointer.
CSS
cursor:pointer;

14. display

Controls how an element appears.
CSS
display:flex;

15. justify-content

Aligns items horizontally.
CSS
justify-content:center;

16. align-items

Aligns items vertically.
CSS
align-items:center;

17. min-height

Sets the minimum height.
CSS
min-height:100vh;

18. transition

Creates animation.
CSS
transition:0.3s;

19. hover Effect

Changes appearance when the mouse is over an element.
CSS
button:hover{
background:red;
}
Styling a Login Form

Example:

CSS
form{
width:350px;
padding:30px;
background:white;
border-radius:10px;
box-shadow:0 0 10px gray;
margin:auto;
}
Styling Input Boxes
CSS
input{
width:100%;
padding:12px;
margin:10px 0;
border:1px solid gray;
border-radius:5px;
}

Styling Buttons

CSS
button{
width:100%;
padding:12px;
background:blue;
color:white;
border:none;
border-radius:5px;
cursor:pointer;
}

Hover effect:

CSS
button:hover{
background:darkblue;
}

Styling the Body

CSS
body{
margin:0;
padding:0;
font-family:Arial;
background:#f4f4f4;
display:flex;
justify-content:center;
align-items:center;
min-height:100vh;
}
